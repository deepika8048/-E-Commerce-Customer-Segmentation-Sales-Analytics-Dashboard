# 🛒 E-Commerce Customer Segmentation & Sales Analytics Dashboard

## 📊 Dashboard Overview

This Power BI dashboard analyzes e-commerce sales performance and customer behavior using RFM (Recency, Frequency, Monetary) segmentation. The project helps businesses identify valuable customers, understand purchasing patterns, monitor revenue trends, and improve customer retention strategies.

---

## 📌 Project Objectives

* Analyze monthly sales performance
* Identify high-value customers
* Perform RFM customer segmentation
* Detect potential customer churn
* Monitor revenue and order trends
* Support data-driven marketing decisions
